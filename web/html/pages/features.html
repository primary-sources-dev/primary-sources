Platform Vision

Primary Sources is more than a digital archive. It is an atomic engine built to reconstruct the past with forensic precision. Below are the capabilities currently live and the visionary features we are architecting.

check_circle

Current Capabilities (Phase 1 Native)

schema Active

Atomic 4NF Engine

Knowledge graph deconstruction. Maps relationships between people, places, and objects across decades.

fact_check Active

Assertion Layer

Multi-witness claim tracking. Handles conflicting primary source accounts without data collisions.

document_scanner Active

Member-Only OCR

High-speed transcription utility supporting PDFs and native archival images with .txt, .md, and high-fidelity .html transcriptions.

travel_explore Active

Hierarchical Browse

Entity-driven discovery system for People, Places, Organizations, and Chronological Events.

analytics Active

Extraction Workbench

Dual-pane review environment featuring Deep Sync coordinate mapping and visual PDF splitting.

tag Active

Forensic Metadata Parser

Automated extraction of archival metadata (RIF, Agency, Date, Author) from document headers and footers.

photo_camera Active

Expanded Media Ingestion

Direct OCR for images (.jpg, .png, .tiff, .webp, .bmp) and iPhone photos (.heic) without conversion.

cake Active

Age-at-Event Badge

Automatically calculates a person's age at the moment of any event using birth record timestamps.

trending_up Active

Inflation Converter

Historical USD to 2026 purchasing power using CPI reference data (1913-2026).

format_quote Active

Auto-Generated Citations

One-click export to NARA, Chicago, MLA, and APA citation formats from source records.

link Active

Entity Matcher

Automated sidecar output mapping OCR text to known IDs using regex-strict and fuzzy-probabilistic logic.

psychology Active

Intelligence Layer

"Psychology" toggle surfacing AI-extracted entities (CIA, FBI, Oswald) directly on archival document scans.

view_quilt Active

Layout Analyzer

Automated classification of document types (FBI 302, NARA) with zone-specific extraction rules.
spellcheck Active

Fuzzy Matching Engine

Levenshtein-based entity linking using RapidFuzz. Resolves OCR errors and name variations with confidence scoring.

Long-term Vision (Advanced Feature Set)

Forensic & Community
  • Conflict Heatmap (#02) Visualize "hot zones" of evidence where sources disagree most.
  • Network Explorer (#03) "Six degrees of separation" visualizer for entity relationships.
  • Achievement Badges (#16) Earn "Expert" status by contributing data and completing quizzes.
  • Infinite Canvas (#18) A massive spatial workbench for pinning and connecting evidence.
Narrative & Simulation
  • POV Shadow Timelines (#05) Filter the world chronology through the eyes of one witness.
  • Witness POV Video (#11) AI recreations of what witnesses saw from specific coordinates.
  • AR Spatial Overlays (#12) Project historical entities onto modern real-world streets.
  • Game Reconstructor (#09) Simulate historical sports games from box scores and logs.
Genealogy & Research
  • Lineage Engine (#22) Track bloodlines and family trees across centuries of records.
  • Ancestral Chronology (#23) Compare family legends against verified primary source data.
  • Self-Service Vault (#20) OCR and analyze your own private documents in the vault.
  • Private Forensic Notes (#19) Encrypted personal notes attached to persistent entities.

Built to Reconstruct Anything

The Primary Sources engine is subject-agnostic. While we are starting with the JFK investigation, the underlying logic can reconstruct cold cases, track family lineages, map cultural movements, or even simulate historical sports seasons from raw data.
