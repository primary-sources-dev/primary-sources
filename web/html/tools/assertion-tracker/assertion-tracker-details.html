Assertion Based Tracking

The research platform's core methodology for identifying, consolidating, organizing, and reviewing massive amounts of historical information into concentrated, source-traced entities with full traceability and contradiction detection.

Source Traceability Contradiction Detection Entity Consolidation Confidence Scoring

What It Does

The Assertion Based Tracking system represents a fundamental shift from traditional historical databases. Instead of storing "facts," the platform records claims (assertions) with full traceability to their original sources. Each assertion is linked to a specific source excerpt (page number, timestamp, context), enabling researchers to instantly verify evidence and identify contradictions.

When processing archival documents, the system automatically extracts entities (people, places, events, objects), consolidates duplicate references, and maps assertions to existing knowledge. If Agent Kesler testified "Yates picked up hitchhiker on Nov 20" but Yates told police "Nov 21," both assertions are preserved with their source excerpts, confidence scores, and contradiction markers.

This methodology transforms unstructured historical documents into a queryable knowledge graph where every claim can be traced to its source, conflicting accounts are explicitly marked, and researchers can instantly see the full evidentiary chain.

Why It Matters

close

Traditional Approach

Wikipedia stores "Lee Harvey Oswald shot President Kennedy" as a fact. Conflicting accounts are buried in talk pages or lost entirely.

check_circle

Assertion-Based Approach

System records: "Warren Commission concluded Oswald shot Kennedy (Warren Report Vol 1, p.19)" AND "HSCA concluded conspiracy probable (HSCA Final Report, p.1)" - both assertions visible, both sourced, contradiction explicit.

How It Works

1

Document Upload → OCR

Archival documents processed through OCR pipeline, classified by type (FBI 302, CIA cable, testimony, etc.)

2

Entity Extraction

Named entity recognition identifies people, places, organizations, objects, and events mentioned in text

3

Assertion Creation

Claims extracted as subject-predicate-object triples: "Yates (subject) PICKED_UP (predicate) Hitchhiker (object)"

4

Source Linking

Each assertion linked to exact source excerpt (document ID, page number, paragraph, timestamp)

5

Conflict Detection

System compares assertions about same subject-predicate pair, flags contradictions with confidence scores

6

Review & Export

Researchers query assertions, review contradictions, trace evidence chains, export citations

Confidence Scoring

Assertions rated CONFIRMED, SUPPORTED, DISPUTED, or UNLIKELY based on source credibility and corroboration.

Evidence-Based
schedule

Time Precision Modeling

Timestamps tagged UNKNOWN, APPROX, EXACT, or RANGE to prevent false precision. "November 1963" ≠ "Nov 22, 1963 at 12:30"

Historiographic
category

Controlled Vocabulary

Event types, predicates, roles use standardized codes (INTERVIEW, SHOT, TESTIFIED_THAT) for consistent querying.

Semantic Consistency

The Assertion Model

Assertions follow subject-predicate-object structure with source traceability:

{
  "assertion_id": "uuid-12345",
  "subject_type": "person",
  "subject_id": "ralph-yates-uuid",
  "predicate": "PICKED_UP",
  "object_type": "person",
  "object_id": "hitchhiker-uuid",
  "assertion_type": "ACTION",
  "confidence": "SUPPORTED",
  "timestamp": "1963-11-20T00:00:00Z",
  "time_precision": "APPROX",
  "source_excerpts": [
    {
      "source_id": "fbi-302-kesler-uuid",
      "page": 2,
      "excerpt": "Yates stated he picked up a hitchhiker...",
      "support_type": "SUPPORTS"
    }
  ]
}

Example Use Cases

Query: "What are all conflicting accounts of when Yates picked up the hitchhiker?"

→ Returns 3 assertions: FBI report says Nov 20, police interview says Nov 21, Yates testimony says "around Thanksgiving" - all with sources.

Query: "Show me every event Ruby participated in where Oswald was also present"

→ Cross-references event_participant table, returns events with both entities, sorted chronologically.

Query: "Which assertions about the Grassy Knoll shooter are contradicted by official sources?"

→ Filters assertions with support_type=CONTRADICTS from Warren Commission/FBI sources.
