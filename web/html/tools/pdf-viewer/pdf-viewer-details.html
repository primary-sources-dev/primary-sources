What It Does

The PDF Viewer provides a seamless document viewing experience integrated directly into the archival research workflow. Researchers can view original scanned documents alongside OCR-extracted text, verifying extraction accuracy before committing records to the database.

With built-in annotation capabilities, researchers can bookmark important pages, add notes, and tag sections for entity extraction — all without leaving the application or switching between tools.

API Usage 

from pdf_viewer import PDFViewer, Annotation

# Load document
viewer = PDFViewer("path/to/document.pdf")

# Navigate pages
viewer.goto_page(5)
text = viewer.get_page_text(5)

# Add annotation
annotation = Annotation(
    page=5,
    type="bookmark",
    note="Key witness testimony begins here"
)
viewer.add_annotation(annotation)

# Search within document
results = viewer.search("Oswald")
for result in results:
    print(f"Page {result.page}: {result.context}")
