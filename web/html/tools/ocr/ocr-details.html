OCR Tool

Optical character recognition engine for archival research. Extract searchable text from scanned documents with batch processing, multi-format support, and integrated quality review.

Tesseract Engine Batch Processing Multi-Format

What It Does

The OCR Tool transforms scanned archival documents into searchable, machine-readable text. Using the Tesseract OCR engine, it processes everything from single-page FBI memos to multi-hundred-page Warren Commission volumes with consistent accuracy.

Researchers can drag-and-drop files or entire archives, queue them for batch processing, and review extracted text before committing to the database — all within a unified workflow designed for high-volume archival ingestion.

Key Features

batch_prediction

Batch Processing

Queue multiple documents for sequential processing. Drop entire ZIP archives and process hundreds of pages automatically.

view_column

Extraction Workbench

Dual-pane review environment featuring Deep Sync coordinate mapping and visual PDF splitting for precise text verification.

auto_fix_high

Image Preprocessing

Automatic deskew, contrast enhancement, and noise reduction optimize scan quality before OCR processing.

spellcheck

Fuzzy Entity Linking

Advanced Levenshtein distance matching using RapidFuzz to resolve OCR character errors and name variations with real-time confidence scoring.

API Usage 

from ocr_processor import OCRProcessor, ProcessingOptions

# Initialize processor
processor = OCRProcessor(engine="tesseract")

# Configure options
options = ProcessingOptions(
    deskew=True,
    enhance_contrast=True,
    language="eng"
)

# Process single document
result = processor.process("document.pdf", options)
print(f"Pages processed: {result.page_count}")
print(f"Text extracted: {len(result.text)} characters")

# Batch process archive
results = processor.process_archive("documents.zip", options)
for doc in results:
    print(f"{doc.filename}: {doc.page_count} pages")
