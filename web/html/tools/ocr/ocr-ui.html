Quick Start

  1. 1. Add Files — Drag PDFs onto the drop zone or click to browse
  2. 2. Configure — Select your backend and output options
  3. 3. Process — Click "Start OCR" and monitor progress in the queue below
  4. 4. Review — Use the "View" button to check results in the PDF viewer

Backend Options

WSL (ocrmypdf)

High-quality OCR using the industry-standard ocrmypdf tool running in Windows Subsystem for Linux.

  • • Best accuracy for archival documents
  • • Creates searchable PDFs with invisible text layer
  • • Supports advanced preprocessing (deskew, clean)
  • • Requires WSL with ocrmypdf installed

Python (pytesseract)

Windows-native OCR using Python's pytesseract wrapper for Tesseract.

  • • Faster setup — no WSL required
  • • Outputs plain text extraction
  • • Good for quick text extraction tasks
  • • Requires Tesseract OCR installed on Windows

Output Formats

picture_as_pdf

Searchable PDF

Original document with invisible text layer. Select and search text while preserving original appearance.

description

Plain Text (.txt)

Raw extracted text with page markers. Useful for full-text search indexing and text analysis.

code

Markdown (.md)

Structured text with headings and formatting preserved. Ideal for research notes and documentation.

Processing Options

straighten

Deskew

Automatically straightens crooked scanned pages. Essential for documents that were fed through a scanner at an angle.

auto_fix_high

Clean

Removes noise, specks, and artifacts from old photocopies. Improves OCR accuracy on degraded documents.

refresh

Force OCR

Re-processes files that already have text layers. Use when existing OCR quality is poor or incomplete.

Tips for Best Results

  • • Batch processing — Queue multiple files and process overnight for large collections
  • • Quality check — Test a few pages before processing an entire collection
  • • Use WSL — For archival documents, WSL backend provides significantly better accuracy
  • • Enable both options — Deskew + Clean together work best on old photocopies
  • • Output directory — Double-click the path to copy it to your clipboard
