================================================================================
PRIMARY SOURCES - SERVER STARTUP GUIDE
================================================================================

Run these commands in Windows PowerShell or Command Prompt (outside of Cursor).

--------------------------------------------------------------------------------
STEP 1: Navigate to Project Root
--------------------------------------------------------------------------------

    cd C:\Users\willh\Desktop\primary-sources

--------------------------------------------------------------------------------
STEP 2: Start Static File Server (Port 8000)
--------------------------------------------------------------------------------

    python -m http.server 8000

Keep this terminal open. Access UI at:
  - http://localhost:8000/docs/ui/classifier-review.html
  - http://localhost:8000/docs/ui/index.html

--------------------------------------------------------------------------------
STEP 3: Start OCR/Feedback Server (Port 5000)
--------------------------------------------------------------------------------

Open a SECOND terminal window, then:

    cd C:\Users\willh\Desktop\primary-sources
    python tools/ocr_server.py

Keep this terminal open. Handles feedback API and OCR processing.

--------------------------------------------------------------------------------
TO STOP SERVERS
--------------------------------------------------------------------------------

Press Ctrl+C in each terminal window.

--------------------------------------------------------------------------------
QUICK REFERENCE
--------------------------------------------------------------------------------

  Server   Port   Command                         Purpose
  ------   ----   -------                         -------
  Static   8000   python -m http.server 8000      Serves HTML, CSS, JS, PDFs
  OCR      5000   python tools/ocr_server.py      Feedback API, OCR processing

CRITICAL: Always run from C:\Users\willh\Desktop\primary-sources
          Do NOT run from any subdirectory.
