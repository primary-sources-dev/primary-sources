Primary Sources leverages the **Material Symbols Outlined** variable font. All icons are served locally from `/web/html/assets/fonts/material-symbols.woff2` to ensure consistent, offline-ready performance.

## Usage Pattern
To use an icon, use a `span` with the `.material-symbols-outlined` class:
```html
<span class="material-symbols-outlined">icon_name</span>
```
