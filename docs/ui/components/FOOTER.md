# Footer Component (`footer.html`)

Standard site-wide footer providing project context and legal information.

## Usage
```html
<div data-component="footer"></div>
```

## Features
- **Project Info**: Brief description of the Primary Sources project.
- **Link Groups**: Organized navigation links for quick access to entities and tools.
- **Legal**: Copyright notice and data attribution.
- **Branding**: Secondary logo placement and theme-aligned colors.

## Integration
- **File**: `web/html/components/footer.html`
- **Component ID**: `data-component="footer"`
- **Styles**: Defined in `main.css`, utilizing `var(--archive-bg)`.
