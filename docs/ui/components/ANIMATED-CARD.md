# Animated Card (`animated-card.html`)

Standardized card component for features and roadmap items with built-in status indicators.

## Usage
```html
<div data-component="animated-card" 
     data-title="On This Day" 
     data-status="live" 
     data-icon="auto_awesome"></div>
```

## Features
- **Visual Branding**: Large background icon display.
- **Status Badges**: Multi-color badges for `queue`, `planned`, and `live` states.
- **Animations**: Hover-triggered scale and opacity transitions.

## Integration
- **File**: `web/html/components/animated-card.html`
- **Data Attributes**: `data-title`, `data-status`, `data-icon`.
- **Pages Using**: `index.html`, `tools-index.html`.
