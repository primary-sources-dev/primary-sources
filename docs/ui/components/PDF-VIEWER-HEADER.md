# PDF Viewer Header (`pdf-viewer-header.html`)

A specialized toolbar for the interactive document viewer.

## Usage
```html
<div data-component="pdf-viewer-header"></div>
```

## Features
- **Document Info**: Displays title and metadata of the active document.
- **Page Controls**: Navigation (prev/next) and direct page number entry.
- **Zoom Controls**: Zoom in/out and Fit-to-Width toggles.
- **Workbench Toggles**: Activates the Intelligence Layer and Extraction Workbench.

## Integration
- **File**: `web/html/components/pdf-viewer-header.html`
- **CSS Dependency**: `assets/css/pdf-viewer.css`
- **Parent Container**: Used specifically in `pdf-viewer-ui.html`.
