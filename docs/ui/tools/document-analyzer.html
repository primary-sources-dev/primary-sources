Intelligent document classification engine that automatically identifies document types from scanned archival materials and extracts structured metadata.

What It Does

The Document Layout Analyzer transforms raw OCR text into structured, classified data. Using weighted fingerprint pattern matching, it identifies the document type and applies zone-specific extraction logic to pull metadata from headers, footers, and body content.

This enables one-click archival ingestion: researchers upload a scanned document and receive a fully classified record with extracted metadata fields ready for database commit.

Key Features

fingerprint

Fingerprint Matching

Weighted regex patterns identify document types by matching unique textual markers in headers and footers.

view_agenda

Zone-Based Extraction

Applies type-specific patterns to header, body, and footer zones for accurate field extraction.

spellcheck

OCR Tolerance

Handles common OCR artifacts including hyphenated line breaks, garbled text, and field office codes.

speed

Confidence Scoring

Each classification includes a confidence score (0-100%) indicating match strength for human review.

API Usage 

from document_classifier import classify_document
from zone_extractor import extract_document

# Classify document type
result = classify_document(ocr_text)
print(f"Type: {result.doc_type.value}")
print(f"Confidence: {result.confidence:.0%}")

# Extract metadata fields
extraction = extract_document(ocr_text)
for field, value in extraction.fields.items():
    print(f"  {field}: {value}")
