Citation Generator

Academic citation formatter for archival research. Generate properly formatted citations from source metadata in Chicago, MLA, APA, and NARA archival styles.

What It Does

The Citation Generator transforms archival source metadata into properly formatted academic citations. Researchers can export citations in four major formats with a single click, ensuring bibliographic accuracy for scholarly publications.

The tool handles complex author name formatting (including government agent designations like "SA"), date conversions across style requirements, and NARA Record Identification Form (RIF) number formatting.

Supported Formats

Chicago 17th Edition

Author. "Title." Source Type, Date. Archive, Reference.

C. Ray Hall. "Interview of Ralph Leon Yates." FBI Interview Report (FD-302), November 26, 1963. NARA, File No. DL 89-43.
MLA 9th Edition

Author. "Title." Source Type, Archive, Date. Reference.

C. Ray Hall. "Interview of Ralph Leon Yates." FBI Interview Report (FD-302), NARA, 26 Nov. 1963. File No. DL 89-43.
APA 7th Edition

Author (Year, Month Day). Title. Source Type. Archive. Reference.

Hall, C. R. (1963, November 26). Interview of Ralph Leon Yates. FBI Interview Report (FD-302). NARA. File No. DL 89-43.
NARA Archival Format

RIF [Number]; [Title]; [Date]; [Archive]

File No. DL 89-43; Interview of Ralph Leon Yates; November 26, 1963; NARA; Author: C. Ray Hall

Key Features

badge

Agent Name Formatting

Automatically handles government agent designations (SA, SSA) and converts to proper academic name formats.

calendar_month

Date Conversion

Converts ISO dates to style-specific formats (Month Day, Year for Chicago; Day Mon. Year for MLA).

numbers

RIF Number Support

Recognizes and formats NARA Record Identification Form numbers (###-#####-#####) for archival citations.

content_copy

One-Click Export

Generate all four citation formats simultaneously from a single source record for easy comparison.

API Usage 

from citation_generator import generate_citation, CitationFormat

source = {
    "title": "Interview of Ralph Leon Yates",
    "author": "SA C. Ray Hall",
    "published_date": "1963-11-26",
    "source_type": "FBI_302",
    "external_ref": "DL 89-43",
    "archive": "NARA"
}

chicago = generate_citation(source, CitationFormat.CHICAGO)
mla = generate_citation(source, CitationFormat.MLA)
apa = generate_citation(source, CitationFormat.APA)
nara = generate_citation(source, CitationFormat.NARA)
