OCR API Reference

REST API documentation for the OCR server. Provides endpoints for document classification, metadata extraction, citation generation, entity matching, and feedback collection.

12 Endpoints JSON REST Flask Backend

Base URL

http://localhost:5000

Start the server with: python tools/ocr_server.py

Classification & Extraction

POST /api/classify

Classify document type from OCR text using fingerprint pattern matching.

Request

{ "text": "FEDERAL BUREAU OF..." }

Response

{
  "doc_type": "FBI_302",
  "confidence": 0.85,
  "confidence_label": "HIGH",
  "matched_patterns": [...]
}
POST /api/extract

Full extraction pipeline: classify document then extract type-specific fields.

Request

{ "text": "OCR text content..." }

Response

{
  "doc_type": "FBI_302",
  "doc_type_confidence": 0.85,
  "fields": {
    "subject_name": {...},
    "file_number": {...}
  }
}
POST /api/parse-metadata

Parse OCR text to extract archival metadata from headers and footers.

Request

{ "text": "OCR text content..." }

Response

{
  "rif_number": {...},
  "agency": {...},
  "date_iso": "1963-11-22",
  "segments": [...]
}

Entity Matching

POST /api/entities

Find known entities (people, places, orgs) in OCR text.

Request

{
  "text": "Lee Harvey Oswald...",
  "include_candidates": true
}

Response

{
  "entities": [...],
  "new_candidates": [...],
  "summary": { "matched": 4 }
}
GET /api/entities/index

Get info about the loaded entity index.

Response

{ "total_entities": 19, "breakdown": { "persons": 5, "places": 8 } }

Citations & Research

POST /api/cite

Generate academic citations in Chicago, MLA, APA, or NARA format.

Request

{
  "source": {
    "title": "Interview Report",
    "author": "SA C. Ray Hall",
    "source_type": "FBI_302"
  },
  "format": "chicago"
}

Response

{
  "citation": "Hall, C. Ray...",
  "format": "chicago"
}
POST /api/inflate

Convert historical USD to modern purchasing power.

Request

{ "amount": 25.00, "year": 1963 }

Response

{
  "converted_amount": 264.71,
  "multiplier": 10.59,
  "formatted": "$25 (1963) = $264.71"
}
GET /api/inflate/multiplier?year=1963

Get inflation multiplier for a specific year.

Response

{ "year": 1963, "target": 2026, "multiplier": 10.59 }

Feedback & Training

POST /api/feedback

Save classification feedback for training data collection.

Request

{
  "page": 42,
  "source": "volume-1.pdf",
  "predictedType": "WC_TESTIMONY",
  "selectedType": "FBI_302",
  "status": "incorrect",
  "textSample": "First 500 chars..."
}

Response

{
  "success": true,
  "totalEntries": 150,
  "summary": {
    "total": 150,
    "correct": 120,
    "incorrect": 30
  }
}
GET /api/feedback

Retrieve all saved feedback entries.

Response

{ "entries": [...], "summary": { "total": 150, "correct": 120, "incorrect": 30 } }
GET /api/feedback/corrections

Get only incorrect classifications grouped by type pair for training analysis.

Response

{
  "corrections": [
    { "predictedType": "WC_TESTIMONY", "selectedType": "FBI_302", "count": 5, "samples": [...] }
  ],
  "total_incorrect": 30
}

OCR Jobs

Method Endpoint Description

POST /api/jobs/{job_id}/start Start processing a queued job
POST /api/jobs/{job_id}/cancel Cancel a running job
GET /api/download/{filename} Download processed file
